# tes-skd-batch3
Modul Aplikasi Tes SKD (Seleksi Kompetensi Dasar)
Aplikasi ini dikembangkan untuk memfasilitasi proses seleksi peserta Magang Nasional Batch III di BPVP Pangkep. Sistem ini memastikan alur pendaftaran dan pengerjaan ujian berjalan secara terintegrasi dan efisien.

Fitur Utama:
- Registrasi Peserta : Validasi data diri peserta.
- Seleksi Formasi: Fitur pemilihan formasi (Formasi 1 & Formasi 2) yang disesuaikan dengan kebutuhan pemagangan.
- Manajemen Ujian: Sistem pengerjaan 110 soal pilihan ganda dengan fitur timer otomatis selama 100 menit untuk menjaga integritas ujian.
- Antarmuka Responsif: Desain antarmuka yang bersih untuk memudahkan navigasi peserta saat berpindah antar soal.

Teknologi & Tools:
- Backend: PHP
- Frontend: CSS
